# Calculadora-de-Circulos-e-Esferas
Este projeto consiste em uma calculadora simples desenvolvida com o objetivo de realizar operações matemáticas relacionadas a figuras geométricas específicas: o círculo e a esfera.

### **A aplicação permite ao usuário calcular:**

* **_Área do círculo;_**
* **_Circunferência do círculo;_**
* **_Volume da esfera._**

A partir do valor do raio informado, o sistema aplica as fórmulas matemáticas correspondentes para retornar os resultados de forma rápida e precisa.

### Conceitos Matemáticos Utilizados

* **_Área do círculo: `π × r²`_**
* **_Circunferência: `2 × π × r`_**
* **_Volume da esfera: `(4/3) × π × r³`_**

### Objetivo do Projeto

Este projeto foi desenvolvido com fins educacionais, visando praticar lógica de programação, aplicação de fórmulas matemáticas e estruturação de um sistema simples.

### Funcionalidades

* **_Entrada do valor do raio;_**
* **_Cálculo automático das operações;_**
* **_Exibição clara dos resultados._**

### Possíveis Melhorias

* **_Interface gráfica mais intuitiva;_**
* **_Validação de entrada de dados;_**
* **_Suporte a outras formas geométricas;_**
* **_Conversão de unidades._**


## Como Executar o Projeto

Ao iniciar o programa, o usuário será guiado por um menu interativo no terminal.

#### O sistema solicitará a escolha de uma operação:

* **_`A` → Calcular a área do círculo_**
* **_`C` → Calcular a circunferência_**
* **_`V` → Calcular o volume da esfera_**

Em seguida, será solicitado o valor do raio.

O programa realizará o cálculo e exibirá o resultado correspondente.

Após a exibição, o sistema perguntará se o usuário deseja reiniciar:

* **_`S` → Reinicia o programa_**
* **_`N` → Encerra o programa_**

Qualquer outro caractere será considerado inválido!
